========================================
  ИНСТРУКЦИЯ ПО РАБОТЕ С KIE AI
========================================

НОВЫЙ ФУНКЦИОНАЛ:
-----------------
Бот теперь поддерживает полную работу с моделями KIE AI:
✅ Просмотр доступных моделей
✅ Выбор модели для генерации
✅ Интерактивный ввод параметров
✅ Генерация контента

КАК ИСПОЛЬЗОВАТЬ:
-----------------

1. ПРОСМОТР МОДЕЛЕЙ
   Команда: /models
   - Показывает список всех доступных моделей KIE AI
   - Каждая модель отображается с описанием
   - Можно выбрать модель прямо из списка

2. НАЧАЛО ГЕНЕРАЦИИ
   Команда: /generate
   - Запускает процесс генерации контента
   - Сначала нужно выбрать модель
   - Затем ввести параметры

3. ПРОЦЕСС ГЕНЕРАЦИИ:
   
   Шаг 1: Выбор модели
   - Используйте кнопки для выбора модели
   - Или команду /models для просмотра списка
   
   Шаг 2: Ввод параметров
   - Бот покажет список необходимых параметров
   - Вводите параметры в формате:
     параметр: значение
   - Или отправьте JSON объект:
     {"param1": "value1", "param2": "value2"}
   - Когда закончите, отправьте /done
   
   Шаг 3: Подтверждение
   - Бот покажет выбранную модель и параметры
   - Подтвердите генерацию кнопкой
   
   Шаг 4: Получение результата
   - Бот сгенерирует контент и отправит результат

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
---------------------

Пример 1: Простая генерация
1. /generate
2. Выберите модель из списка
3. Введите текст (если модель требует только текст)
4. /done
5. Подтвердите генерацию

Пример 2: Генерация с параметрами
1. /models
2. Выберите модель (например, для генерации изображений)
3. Введите параметры:
   prompt: красивое закатное небо
   width: 1024
   height: 768
4. /done
5. Подтвердите генерацию

КОМАНДЫ:
--------
/start - Начать работу с ботом
/help - Справка по командам
/models - Показать доступные модели
/generate - Начать генерацию контента
/cancel - Отменить текущую операцию
/search [запрос] - Поиск в базе знаний
/ask [вопрос] - Задать вопрос
/add [знание] - Добавить знание

НАСТРОЙКА:
----------
Для работы с KIE AI необходимо в файле .env указать:

KIE_API_URL=https://api.kie.ai
KIE_API_KEY=ваш_api_ключ

Без этих настроек бот будет работать только с базой знаний.

ОТМЕНА ОПЕРАЦИИ:
----------------
В любой момент можно отменить операцию командой /cancel
или нажав кнопку "Отмена"
