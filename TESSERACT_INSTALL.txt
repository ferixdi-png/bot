========================================
ИНСТРУКЦИЯ ПО УСТАНОВКЕ TESSERACT OCR
========================================

Tesseract OCR необходим для автоматической проверки скриншотов оплаты.

ШАГ 1: Установка Tesseract OCR для Windows
========================================

1. Скачайте установщик:
   - Откройте: https://github.com/UB-Mannheim/tesseract/wiki
   - Скачайте последнюю версию для Windows (например: tesseract-ocr-w64-setup-5.5.0.20241201.exe)

2. Запустите установщик:
   - Дважды кликните на скачанный .exe файл
   - Следуйте инструкциям установщика

3. ВАЖНО при установке:
   - Выберите путь установки (по умолчанию: C:\Program Files\Tesseract-OCR)
   - Обязательно выберите языки:
     ✓ English (обязательно)
     ✓ Russian (для распознавания русского текста)
   - Остальные языки опционально

4. После установки:
   - Tesseract будет установлен в: C:\Program Files\Tesseract-OCR
   - Исполняемый файл: C:\Program Files\Tesseract-OCR\tesseract.exe

ШАГ 2: Настройка пути (если нужно)
========================================

Если Tesseract установлен не в стандартное место, добавьте путь в код:

В файле bot_kie.py после импортов добавьте:

import pytesseract
pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'

(Замените путь на ваш, если установили в другое место)

ШАГ 3: Проверка установки
========================================

Откройте командную строку (cmd) и выполните:

tesseract --version

Должно показать версию Tesseract (например: tesseract 5.5.0)

ШАГ 4: Перезапуск бота
========================================

После установки Tesseract:
1. Закройте бота (если запущен)
2. Запустите бота заново
3. Система автоматически обнаружит Tesseract и начнет проверять скриншоты

ЧТО ДЕЛАЕТ TESSERACT:
========================================

После установки Tesseract система будет:
✅ Проверять сумму перевода на скриншоте
✅ Проверять номер телефона (если указан в .env)
✅ Искать признаки платежа (ключевые слова)
✅ Защищать от поддельных скриншотов

ЕСЛИ TESSERACT НЕ УСТАНОВЛЕН:
========================================

Система будет работать, но:
⚠️ Баланс будет начисляться автоматически без проверки
⚠️ Защита от поддельных скриншотов будет отключена
⚠️ Рекомендуется установить Tesseract для безопасности

ПРОБЛЕМЫ И РЕШЕНИЯ:
========================================

Проблема: "tesseract is not installed"
Решение: Установите Tesseract OCR (см. ШАГ 1)

Проблема: "Failed to load language 'rus'"
Решение: При установке выберите русский язык

Проблема: "tesseract not found"
Решение: Добавьте путь в код (см. ШАГ 2) или добавьте Tesseract в PATH системы
